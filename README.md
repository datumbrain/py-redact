# py-redact

Document redaction.

## Make Virtual Env.

```
mkvirtualenv -p python3.7 py-redact
```

## Use `workon` to Use virtualenv Python Interpreter

```
workon py-redact
```


## Install Requirements

```
pip install -r requirements.txt
```

## Run

```
python main.py <input_file_path> <output_file_path>
```
